問題３の解答プログラムは，このディレクトリに作成しなさい．
解答プログラムのソースファイル名は Overview Sheet の指示に従い
	[受験番号]_t3.[拡張子]
としなさい．
受験番号のアルファベットは全て大文字です．
拡張子は，C言語は "c", C++は "cpp", JAVA は "java" です．
例えば，受験番号 J0800001AX の人が C++ で解答した場合のソースのファイル名は
    J0800001AX_t3.cpp
となります．

スクリプトの使用方法：
compile_t3		採点時と同じ方法でコンパイルするスクリプト
			　使用例
			　$ ./compile_t3 [受験番号] [言語指定]
run_t3			指定した入力データ例に対して実行するスクリプト
			　注）先に compile_t3 でコンパイルしておくこと
			　使用例
			　$ ./run_t3 [受験番号] [言語指定] [入力データ例番号]
eval_t3			指定した入力データ例に対して実行し，
			その出力を正解と比較するスクリプト
			　注）先に compile_t3 でコンパイルしておくこと
			　使用例
			　$ ./eval_t3 [受験番号] [言語] [入力データ例番号]
			  正解時は "Correct" と，不正解時は "Wrong" と表示

注意：
・　受験番号のアルファベットは全て大文字で指定すること
・　言語の指定は，C言語は "c", C++は "cpp", JAVA は "java" とすること
・　run_t3 および eval_t3 を実行途中で終了させたい場合は，Ctrl-C
